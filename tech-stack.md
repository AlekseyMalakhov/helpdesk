## Tech Stack

### Frontend

- **React with TypeScript** — UI framework
- **Tailwind CSS + shadcn/ui** — styling and component library
- **React Router**

### Backend

- **Node.js with Express.js and TypeScript** — REST API server
- **Prisma** — type-safe ORM
- **PostgreSQL** — relational database

### Authentication

- **Database sessions** — sessions stored in PostgreSQL

### AI

- **Anthropic Claude API** — ticket classification, summaries, and suggested replies

### Email

- **SendGrid or Mailgun** — for outbound replies and inbound webhooks

### Deployment

- Docker + cloud provider (AWS, Railway, Fly, etc)
